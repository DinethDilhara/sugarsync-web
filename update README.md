With a focus on simplicity and usability, SugarSync enables users to track their blood sugar levels, medication intake, meals, and other important health metrics. ## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/DinethDilhara/sugarsync-web.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd sugarsync-web/SugarSync-Web/
   ```

3. **Install dependencies** using Yarn:
   ```bash
   yarn install
   ```

4. **Run the application**:
   ```bash
   yarn start
   ```

5. Open your browser and navigate to `http://localhost:3000` to view the app. ğŸŒ
